# 4 Different Design Maps of Seattle

This project showcases four different tile map designs of Seattle, implemented using **Mapbox GL JS**. Users can toggle between the layers using the menu on the right and explore the city’s pickleball court locations in combination with basemaps and a themed map.

---

## Map Layers / Tile Sets

1. **Basemap**  
   - Description: A clean monochrome basemap providing geographic context of Seattle.  
   - Zoom levels: 10–14  
   - Examined area: City of Seattle

2. **Pickleball**  
   - Description: Displays the locations of pickleball courts throughout Seattle.  
   - Zoom levels: 10–14  
   - Examined area: City of Seattle

3. **Combined**  
   - Description: Overlay of pickleball court locations on top of the basemap for better spatial context.  
   - Zoom levels: 10–14  
   - Examined area: City of Seattle

4. **Theme**  
   - Description: A themed map highlighting Asian Heritage in Seattle, with colors, icons, and labels reflecting this theme.  
   - Zoom levels: 10–14  
   - Examined area: City of Seattle

---

## Data Source

- **Basemap:** Mapbox Light style  
- **Pickleball Courts:** [Seattle GeoData Pickleball Courts Dataset]([https://data.seattle.gov/](https://data-seattlecitygis.opendata.arcgis.com/datasets/SeattleCityGIS::pickleball-courts/explore?location=47.616431%2C-122.329550%2C11))  
- **Theme Map:** Custom map design based on Asian Heritage
